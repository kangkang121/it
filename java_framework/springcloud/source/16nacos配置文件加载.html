                  二、BootstrapApplicationListener

                  属于springcloud中的自动配置监听器之一，这个就是可以做远程配置的。

                  他排序在ConfigFileApplicationListener前面的，因为ConfigFileApplicationListener会进行文件的加载，如果我想添加一些新的配置文件，就可以在BootstrapApplicationListener中添加，比如bootstrap配置文件就是这么做的。
