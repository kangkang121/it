                                      一、UnpooledHeapByteBuf

                                      介绍

                                      引用计数缓冲区的子类，非池化的堆缓冲区，也就是内存在java堆里，内部其实就是分配了一个数组。这里还有个字节序的概念，也就是说字节是放在地址的高低位的区别。简单的说下，主要分两种，一种是小端字节序，也就是一般内存存放数据的，高字节放在高位地址，低地址放低位地址，而大端刚好是相反，一般用于网络通信获取的字节数据。比如16位数据是0xf123放在字长为2个字节的地址里，如果小端就是高位放f1低位放23，如果是大端就是高位放23低位放f1 。好像挺麻烦的，不就是个存放么，为什么不统一呢，这个貌似是历史遗留原因。
